# TaskFlow — Telegram Mini App для управления задачами

**TaskFlow** — это современное веб-приложение (Telegram Mini App), разработанное для автоматизации взаимодействия между администраторами и исполнителями (воркерами). Приложение позволяет создавать задачи, управлять сменами и отслеживать выполнение работ в режиме реального времени.

---

## 🚀 Основные возможности

### Для администраторов (Admin)
- **Панель управления**: Обзор всех активных, завершенных и отмененных задач.
- **Создание задач**: Удобная форма для публикации новых заявок (название, описание, адрес, время начала/окончания, оплата, количество требуемых сотрудников).
- **Управление заявками**: Просмотр списка воркеров, откликнувшихся на задачу, и подтверждение их участия.
- **База сотрудников**: Список всех зарегистрированных пользователей с их специализациями.

### Для исполнителей (Executor / Worker)
- **Лента вакансий**: Просмотр доступных задач с подробной информацией (местоположение, оплата, график).
- **Отклик на задачи**: Возможность подать заявку на участие в один клик.
- **Мои смены**: Персональный список подтвержденных задач для удобного планирования своего графика.
- **Профиль**: Информация о себе, специализация и статус верификации.

## 🛠 Технологический стек

- **Frontend**: [React 19](https://react.dev/), [TypeScript](https://www.typescriptlang.org/), [Vite](https://vitejs.dev/)
- **Backend & DB**: [Supabase](https://supabase.com/) (PostgreSQL)
- **Интеграция**: [Telegram Mini Apps SDK](https://github.com/Telegram-Mini-Apps/telegram-apps) (`@twa-dev/sdk`)
- **Стилизация**: [Tailwind CSS](https://tailwindcss.com/)
- **Анимации**: [Framer Motion](https://www.framer.com/motion/)
- **Иконки**: [Lucide React](https://lucide.dev/)

## ⚙️ Установка и запуск

1. **Клонируйте репозиторий**:
   ```bash
   git clone <repository-url>
   cd TaskFlow
   ```

2. **Установите зависимости**:
   ```bash
   npm install --legacy-peer-deps
   ```

3. **Настройте переменные окружения**:
   Создайте файл `.env` в корне проекта и добавьте ваши ключи Supabase:
   ```env
   VITE_SUPABASE_URL=ваш_url
   VITE_SUPABASE_ANON_KEY=ваш_ключ
   ```

4. **Запустите сервер для разработки**:
   ```bash
   npm run dev
   ```

5. **Развертывание базы данных**:
   Используйте SQL-скрипт `supabase_schema.sql` для создания необходимых таблиц и индексов в вашем проекте Supabase.
